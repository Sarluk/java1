Закрытие файла высвобождает выделенные для него системные ресурсы, позволяя использовать их для других
файлов. Неудачный исход закрытия файла может привести к "утечкам памяти", поскольку неиспользуемые
ресурсы оперативной памяти останутся выделенными. Для закрытия файланеобходимо вызвать метод close().
Чтобы прочитать данные из файла, можно воспользоваться версией метода read(), определенной в классе
FileInputStream. Всякий раз, когда вызывается метод read(), он выполняет чтение одного байта из файла и
возвращает его в виде целочисленного значения. А если достигнуr конец файла, то возвращается значение - 1 .
В приведенном ниже примере программы метод read() применяется для ввода из файла, содержащего текст в
коде ASCII, который затем выводится на экран.
